    0x0::royalty

    Module of Collection Royalty domain.

    It allows Creators to define a royalty strategy for their collections. It exposes public functions to be used by the Type-exporting Collection type module (e.g. Suimarines) in order to calculate the royalty given a amount. This module relies on the type-exporting module as an oracle for the execution price of the trade.
